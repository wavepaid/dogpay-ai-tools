DogPay Developers

设置您的 AI 工具以集成 DogPay。

在您的项目文件夹中运行此命令。

claude mcp add dogpay-docs -- npx -y @dogpay/ai-docs

添加到 Cursor 的 MCP 配置。

{
  "mcpServers": {
    "dogpay": {
      "command": "npx",
      "args": ["-y", "@dogpay/ai-docs"]
    }
  }
}

在您的项目文件夹中运行此命令。

codex plugin marketplace add wavepaid/dogpay-ai-tools
codex plugin add dogpay@dogpay

在 GPT Builder 的 Actions 中导入 OpenAPI 文件。

outputs/openapi/dogpay-payments.openapi.json

发布 npm 包后可直接运行。

npx -y @dogpay/ai-docs --print codex
Codex Plugin 安装后自动启用 DogPay API 接入规则、鉴权缓存、支付订单和 webhook 处理约束。
MCP / npx Claude Code 与 Cursor 通过 MCP 工具读取 DogPay 端点、搜索文档并生成代码片段。
OpenAPI Action ChatGPT 可导入支付相关 OpenAPI 文件，使用 sandbox 或生产环境调用。
